[2020/07/11] Githubにコミットする方法 備忘録

何故かDiscordに書いた備忘録が消されてしまったのでここに書きます
以下のようにやれば出来るはずです
1. git clone Githubのリポジトリ
例: git clone https://github.com/WinLinux1028/STK1.git
2. cd クローンされたリポジトリ
例: cd STK1
3. いろいろ作業する
4. git add *
5. git commit -a -m "コミット名"
6. git push origin master
6の実行後､ユーザー名(例:WinLinux1028)とパスワードを尋ねられるので､それを入力します
